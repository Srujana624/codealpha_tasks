➢INTERNSHIP TITLE:Data Science Internship – Machine Learning & Data Analysis Projects

➢PROJECT DESCRIPTION:
This project consists of four real-world data science tasks that demonstrate core concepts of machine learning, data analysis, and visualization using Python. The tasks cover classification, regression, and exploratory data analysis techniques applied to practical datasets.

TASK 1: Iris Flower Classification
This task focuses on building a machine learning classification model to predict the species of Iris flowers (Setosa, Versicolor, Virginica) based on their physical measurements.
•Used Scikit-learn’s built-in dataset
•Applied K-Nearest Neighbors (KNN) algorithm
•Split data into training and testing sets
•Evaluated model performance using accuracy and classification report

TASK 2: Unemployment Analysis with Python
This task involves analyzing unemployment data to understand trends and patterns, especially during the Covid-19 period.
•Performed data cleaning and preprocessing
•Visualized unemployment trends over time
•Analyzed the impact of Covid-19 on unemployment rates
•Identified patterns and insights using graphs

TASK 3: Car Price Prediction with Machine Learning
This task aims to predict car prices based on features like brand, mileage, horsepower, etc.
•Cleaned and preprocessed dataset
•Converted categorical data into numerical format
•Used Linear Regression model for prediction
•Evaluated performance using Mean Absolute Error (MAE)

TASK 4: Sales Prediction using Python
This task predicts future sales based on advertising expenditures across different platforms.
•Performed data cleaning and feature selection
•Used regression model for prediction
•Analyzed relationship between advertising and sales
•Visualized results using plots
